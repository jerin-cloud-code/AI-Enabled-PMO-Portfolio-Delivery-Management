# Contributing Guidelines — ai-enabled-portfolio-pmo

Thank you for exploring `ai-enabled-portfolio-pmo`!

## Contribution Workflow
1. Fork the repository and create a feature branch (`git checkout -b feature/my-feature`).
2. Ensure all data generated remains 100% synthetic (no real company/person names).
3. Add unit tests in `tests/` for any new functionality.
4. Run `python scripts/run_all.py` to verify full build and test suite pass.
5. Submit a Pull Request.
