# LinkedIn Project Description

## Title: AI-Enabled Portfolio PMO System (GitHub Demonstration)

**Description:**
I designed and developed `ai-enabled-portfolio-pmo`, a public portfolio demonstration project showing how modern PMO functions can integrate AI assistance, automated data validation, and reproducible executive reporting into standard governance workflows.

**Key Highlights:**
1. **Jira-Style Data Management:** Automated ingestion and normalization of enterprise portfolio data across Technology, Data, AI, Cyber Security, and Fraud Prevention portfolios.
2. **Automated Data Quality:** Validation engine detecting stale updates, missing owners, and budget overruns with detailed exception reporting.
3. **AI Assistance with Human Oversight:** Drafts portfolio summaries offline (or via pluggable LLM) backed by a 6-part factual validation taxonomy and human review workflow.
4. **Executive Deliverables:** One-click generation of One-Page Executive Views, Excel Workbooks, 8-Slide PowerPoint Governance Packs, and HTML Dashboards.
5. **Demand Shaping & Deduplication:** TF-IDF text similarity scoring to detect duplicate initiative proposals during front-door intake.
