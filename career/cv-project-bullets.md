# Recommended CV Bullet Points — AI Portfolio PMO Project

## Section: Key Projects / Technical Demonstrations

### Project: AI-Enabled Portfolio PMO System (GitHub Demonstration Project)
- **Built an automated Python-based PMO governance and reporting suite** processing Jira-style CSV/JSON data across 50 synthetic initiatives spanning Technology, Data, AI, Cyber, and Fraud portfolios.
- **Implemented an automated data-quality validation engine** enforcing 15 business rules to detect stale records, missing sponsors, orphan dependencies, and budget spend overruns (>15%).
- **Developed an AI narrative summariser with factual assurance controls** (NIST AI RMF aligned) that drafts executive updates and cross-checks claims against source metrics before publication.
- **Designed automated deliverable generators** producing one-page executive views, multi-tab Excel workbooks, 8-slide PowerPoint packs, and static HTML dashboards with print optimization.
- **Created an TF-IDF similarity model for front-door demand intake** to detect overlapping initiative proposals during shaping reviews.
