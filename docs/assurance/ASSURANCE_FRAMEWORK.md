# AI Factual Assurance Framework (NIST AI RMF Aligned)

## Factual Validation Taxonomy
1. `VAL-01` (`status_match`): Ground-truth RAG status verification.
2. `VAL-02` (`count_match`): Numerical count reconciliation.
3. `VAL-03` (`date_consistency`): Milestone date verification.
4. `VAL-04` (`rag_consistency`): RAG color alignment with spend variance.
5. `VAL-05` (`named_entity_match`): Initiative entity validation.
6. `VAL-06` (`completeness_check`): Structural coverage check.
